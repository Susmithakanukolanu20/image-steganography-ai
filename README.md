\# Image Steganography with AI Speech



\## Description

This project hides a character inside a grayscale 128×128 image using LSB steganography.

The hidden character is retrieved using reverse steganography and converted to speech

using AI Text-to-Speech.



\## Features

\- Grayscale image conversion

\- Resize to 128×128

\- Hide \& retrieve character

\- AI speech output



\## How to Run

```bash

python steg\_app.py
